Haroonga
===

[![Build Status](https://travis-ci.org/cosmo0920/haroonga.png?branch=master)](https://travis-ci.org/cosmo0920/haroonga)

Low level Haskell binding for Groonga.

## dependencies

* groonga 3.1.1+
* hsc2hs
* and some cabal packages

## export module(s)

* Binding.Groonga
* Binding.Groonga.SimpleAPI
